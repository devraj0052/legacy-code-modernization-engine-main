# System Architecture

The Legacy Code Modernization Engine consists of three main components.

1. User Interface
Built using Streamlit to allow users to enter legacy code.

2. Code Modernization Engine
Python module that processes legacy code patterns and converts them into modern equivalents.

3. Output Module
Displays improved code with better formatting and modern syntax.

Workflow:

User Input → Streamlit Interface → Modernization Engine → Modern Code Output
